# 🧠 Cognitive Aging Brain Analysis

This project explores the relationship between brain structure and cognitive function in aging populations, combining statistical analysis and basic neuroimaging concepts.

---

## 🔍 Research Question

How do structural brain changes relate to cognitive decline, and how can these findings be interpreted using functional neuroimaging frameworks?

---

## ✅ What I Have Done

- Conducted basic statistical analyses in R (data cleaning, descriptive statistics, linear regression)
- Explored fMRI data structure using Nilearn (4D data, voxel-wise time series)
- Interpreted event timing (onset, duration) and visualized BOLD signals
- Implemented a simple GLM to examine stimulus-related brain activation
- Gained conceptual understanding of HRF, design matrix, and contrast

---

## 🚀 What I Plan to Do

- Relate structural brain measures to cognitive performance using regression models
- Perform region-level (ROI) analysis for clearer interpretation
- Refine neuroimaging analysis (e.g., masking, model specification)
- Integrate results into a coherent brain–cognition interpretation
